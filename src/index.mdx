# React Puzzle Game

## Installation

If you use `npm`

```bash
npm install --save react-puzzle-game
```

or with `yarn`

```bash
yarn add react-puzzle-game
```

## Main components

- [`PuzzleGame`](/puzzle-game): the main wrapper
- [`Puzzle`](/puzzle): the sliced, mixed and clickable image
